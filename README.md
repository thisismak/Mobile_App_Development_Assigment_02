### 功能操作說明
網站功能操作說明
1. 首頁 (index.html)
功能概述
- 頁面載入動畫：頁面初次載入時顯示一個旋轉的載入動畫（藍色圓圈與「載入中...」文字），持續1秒後隱藏，隨後顯示主內容。
- 主內容展示：顯示歡迎標題（「歡迎探索開源硬體」）和副標題（「發現並體驗創新的硬體解決方案」），以及一個「開始探索」按鈕。
- 背景動畫：背景包含漂浮的圓形圖形，增添視覺動態效果。
- 頁面切換動畫：點擊「開始探索」按鈕後，觸發圓形擴展轉場動畫，並伴隨粒子效果，隨後跳轉至 `apps.html`。
- 粒子效果：按鈕點擊時，會在按鈕中心生成10個白色小圓點，向四周散開並逐漸消失。

操作說明
1. 進入網站：
   - 打開 `index.html`，頁面會顯示載入動畫（藍色旋轉圓圈和「載入中...」文字）。
   - 1秒後，動畫消失，主內容（標題、副標題和按鈕）淡入顯示。
2. 瀏覽背景效果：
   - 注意背景中的三個白色圓形圖形，它們會緩慢上下漂浮。
3. 進入硬體清單頁面：
   - 點擊「開始探索」按鈕（帶有火箭圖標）。
   - 按鈕中心會觸發藍色圓形擴展動畫，同時生成粒子效果（白色小圓點散開）。
   - 0.8秒後，頁面自動跳轉至 `apps.html`。

 2. 硬體清單頁面 (apps.html)
功能概述
- 硬體清單展示：顯示20個開源硬體項目的卡片，包括名稱、類型、功能、描述、圖片和YouTube影片。
- 視圖切換：支援列表視圖（單列）和網格視圖（多列自適應）。
- 搜尋功能：透過搜尋欄可根據硬體名稱或功能進行篩選。
- 分類篩選：可選擇「微控制器」、「感測器」或「通訊模組」進行分類篩選（支援多選）。
- 標籤篩選：可根據標籤（如「Wi-Fi」、「藍牙」、「低功耗」等）篩選硬體（支援多選）。
- 排序功能：可根據名稱、類型或功能進行升冪或降冪排序。
- 統計圖表：頁面底部顯示一個圓環圖，展示當前篩選結果中硬體類型（微控制器、感測器、通訊模組）的分佈情況，篩選或排序時圖表會動態更新。
- 卡片動畫：每次更新清單時，卡片會以淡入和上移動畫顯示。

操作說明
1. 進入頁面：
   - 從首頁點擊「開始探索」後，進入 `apps.html`，顯示硬體清單（預設為列表視圖）。
   - 頁面頂部有工具列（標題和視圖切換按鈕），下方有搜尋欄、分類選擇、標籤選擇和排序選項。
   - 清單下方顯示圓環圖，展示硬體類型分佈（初始為全部20項硬體的分佈）。

2. 切換視圖：
   - 列表視圖：
     - 點擊工具列右側的「列表」圖標（`list-outline`）。
     - 硬體卡片以單列垂直排列，每張卡片寬度最大600px，居中顯示。
   - 網格視圖：
     - 點擊工具列右側的「網格」圖標（`grid-outline`）。
     - 硬體卡片以多列排列（每列至少300px寬，根據螢幕大小自動調整列數）。

3. 搜尋硬體：
   - 在搜尋欄（顯示「搜尋產品名稱或功能...」）輸入關鍵字（如「Arduino」、「Wi-Fi」）。
   - 清單即時更新，僅顯示名稱或功能包含關鍵字的硬體。
   - 若無符合結果，顯示「無符合條件的硬體」訊息。
   - 清空搜尋欄（點擊搜尋欄右側的「x」），恢復顯示所有符合其他篩選條件的硬體。

4. 篩選硬體：
   - 按分類篩選：
     - 點擊「分類」選擇框（預設顯示「選擇分類（可多選）」）。
     - 在彈出的選單中勾選「微控制器」、「感測器」或「通訊模組」（可多選）。
     - 點擊「確定」，清單更新為僅顯示選中類型的硬體。
     - 取消所有選擇（點擊「取消」或清空選擇），恢復顯示所有類型。
   - 按標籤篩選：
     - 點擊「標籤」選擇框（預設顯示「選擇標籤（可多選）」）。
     - 在彈出的選單中勾選標籤（如「Wi-Fi」、「藍牙」、「低功耗」等，可多選）。
     - 點擊「確定」，清單更新為僅顯示功能包含所有選中標籤的硬體。
     - 取消所有選擇，恢復顯示所有標籤的硬體。

5. 排序硬體：
   - 選擇排序欄位：
     - 點擊「排序欄位」選擇框（預設顯示「選擇排序欄位」）。
     - 選擇「名稱」、「類型」或「功能」。
   - 選擇排序方向：
     - 在「排序欄位」旁邊選擇「升冪」（由A到Z）或「降冪」（由Z到A）。
   - 清單根據所選欄位和方向重新排列，更新後卡片以動畫效果顯示。
   - 若未選擇排序欄位，清單保持篩選後的預設順序。

6. 查看硬體詳情：
   - 每個硬體卡片包含：
     - 圖片：硬體的展示圖片（點擊無互動）。
     - 名稱：硬體名稱（如「Arduino Uno」）。
     - 類型：硬體類型（如「微控制器」）。
     - 功能：硬體主要功能描述（如「易於使用，支援多種外設」）。
     - 說明：硬體簡要介紹（如「適合初學者的開源微控制器板。」）。
     - 標籤：顯示類型和功能的標籤（如「微控制器」、「易於使用」）。
     - 影片：嵌入的YouTube影片，介紹硬體（可直接播放）。
   - 點擊影片區域可播放、暫停或全螢幕觀看。

7. 查看統計圖表：
   - 滾動至清單底部，查看圓環圖（標題為「硬體類型分佈」）。
   - 圖表顯示當前清單中各類型（微控制器、感測器、通訊模組）的數量比例。
   - 每次搜尋、篩選或排序後，圖表會動態更新以反映當前清單的分佈。
   - 圖表顏色固定：微控制器（藍色）、感測器（紅色）、通訊模組（淺藍色）。

 注意事項
- 載入時間：首頁的載入動畫固定為1秒，若網路較慢，圖片或影片可能延遲顯示。
- 篩選邏輯：
  - 搜尋：匹配名稱或功能，忽略大小寫。
  - 分類：僅顯示選中類型的硬體，多選為「或」邏輯。
  - 標籤：僅顯示包含所有選中標籤的硬體，為「且」邏輯。
- 排序邏輯：排序基於文字的字母順序（忽略大小寫），若未選擇排序欄位，則保持數據原始順序。
- 影片播放：YouTube影片需要網路連線，確保設備可訪問YouTube。
- 響應式設計：
  - 網格視圖會根據螢幕寬度自動調整列數。
  - 手機或窄螢幕上，卡片和圖表會適應螢幕大小。
- 錯誤處理：
  - 若初始化失敗（如DOM元素缺失），頁面會顯示錯誤訊息（「應用程式載入失敗，請重新整理」）。
  - 若篩選結果為空，顯示「無符合條件的硬體」。

5. 潛在問題與改進建議

5.1 功能問題
- 固定載入時間（首頁）：
  - 問題：載入動畫固定為1秒，若資源載入過慢（如低速網路），可能導致內容閃現。
  - 建議：改用動態檢測（如 `window.onload` 或資源載入完成事件）控制動畫結束。
- 影片載入（硬體清單）：
  - 問題：20個YouTube影片同時嵌入可能增加初始載入時間，尤其在低速網路下。
  - 建議：實現延遲載入（lazy loading），僅在卡片進入視窗時載入影片。
- 搜尋靈敏度：
  - 問題：搜尋僅匹配名稱和功能，未涵蓋描述或標籤，可能遺漏相關結果。
  - 建議：擴展搜尋範圍至描述和標籤，或提供高級搜尋選項。
- 錯誤提示：
  - 問題：錯誤提示（「應用程式載入失敗」）僅顯示在主內容區域，未提供重試機制。
  - 建議：添加重試按鈕或更詳細的錯誤訊息（如網路斷線提示）。

5.2 性能問題
- 圖片優化：
  - 問題：硬體卡片圖片（`photo/*.jpg`）未壓縮，影響低速網路載入。
  - 建議：使用WebP格式或壓縮圖片，並添加 `loading="lazy"` 屬性。
- 粒子效果：
  - 問題：低端設備上粒子效果可能導致短暫幀率下降（50fps）。
  - 建議：減少粒子數量（從10個減至5個）或根據設備性能動態調整。
- 數據規模：
  - 問題：當前20項數據性能良好，但若硬體清單擴展至數百項，可能出現延遲。
  - 建議：實現分頁或無限滾動，僅渲染可見卡片。

5.3 相容性問題
- 舊版瀏覽器：
  - 問題：部分舊版瀏覽器（如舊版Edge）對CSS動畫（如 `transform`）支援不佳。
  - 建議：添加CSS前綴（透過Autoprefixer）或降級動畫效果。
- 無網路場景：
  - 問題：YouTube影片和CDN資源（Ionic, Chart.js）依賴網路，斷網時頁面功能受限。
  - 建議：提供離線提示，或將關鍵資源本地化。

5.4 使用殞能性問題
- 無鍵盤支援：
  - 問題：鍵盤導航（Tab鍵）在硬體清單頁面的篩選選項（分類、標籤、排序）上不夠流暢。
  - 建議：為 `ion-select` 和 `ion-radio` 添加 `tabindex` 並優化焦點樣式。

### 遇到的問題和解決方法

首頁 (index.html)
1. 問題：載入動畫固定1秒，資源慢載時內容閃現。
 - 解決：用 window.onload 取代 setTimeout 控制動畫。
2. 問題：粒子效果在低端設備幀率下降（50fps）。
 - 解決：減少粒子數至5個，縮短動畫至0.3-0.6秒。
3. 問題：舊版瀏覽器轉場動畫閃爍。
 - 解決：添加CSS前綴和 will-change 優化渲染。
4. 問題：鍵盤無法聚焦「開始探索」按鈕。
 - 解決：添加 tabindex="0" 和焦點樣式。

硬體清單頁面 (apps.html)
1. 問題：20個影片同時載入，延長頁面載入時間。
 - 解決：用IntersectionObserver實現影片延遲載入。
2. 問題：圖片未壓縮，增加低速網路載入時間。
 - 解決：轉為WebP格式，添加 loading="lazy"。
3. 問題：搜尋未涵蓋描述，遺漏相關結果。
 - 解決：擴展搜尋範圍至描述和類型。
4. 問題：篩選選項鍵盤導航不直觀。
 - 解決：為 ion-select 添加焦點樣式和 tabindex。
5. 問題：小螢幕上圓環圖標籤可能重疊。
 - 解決：設置圖表最大寬度，縮小字體和間距。
6. 問題：斷網時影片顯示空白框架，無提示。
 - 解決：添加離線檢測，顯示「影片需網路連線」。
7. 問題：錯誤提示僅顯示「載入失敗」，無重試。
 - 解決：添加重試按鈕，點擊重新載入頁面。
8. 問題：數百項數據可能導致渲染延遲。
 - 解決：實現分頁，每頁渲染10項數據。
